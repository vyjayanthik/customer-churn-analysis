# Project Overview
The goal of this project is to understand the factors that influence customer churn and provide insights to reduce customer loss.

# Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

# Key Analysis
- Overall churn rate calculation
- Churn by contract type
- Churn by tenure
- Churn by monthly charges
- Customer segmentation by price level

# Key Insights
- Customers on month-to-month contracts show the highest churn
- New customers are more likely to leave
- Higher monthly charges increase churn probability
- Long-term customers tend to stay

# Conclusion
Customer churn is mainly influenced by contract type, tenure, and monthly charges. Businesses should encourage long-term contracts and improve retention strategies for new customers.
